安装和配置

  • 安装git hexo
    npm install -g hexo-cli
  • 初始化本地博客项目
  • 安装依赖
  • 切换到项目目录后运行：
    npm install
  • 生成默认文件结构

新增文章

1、在 source/_posts 目录中新增或编辑文章，或通过命令创建新文章：
hexo new "文章标题"
2、运行以下命令将 Markdown 转换为静态网页：
hexo generate
3、本地预览
hexo server

展示到git
hexo clean && hexo generate && hexo deploy
